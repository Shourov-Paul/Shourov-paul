## 🚀 Featured Projects by Shourov Paul

### 🌞 Solar Panel Monitor (ESP32)
- Real-time monitoring of voltage, current, power, and tilt angle using MPU6050 & INA3221 sensors.
- Features an OLED display and a Web dashboard with 3D visualization.

### 📊 ESP32 Web Server with Real-Time Charts
- Live plotting of sensor data using Highcharts served by an Async web server for smooth performance.
- Features customizable thresholds with alert functionalities.

### 💧 Ultrafloat (Water Level Controller)
- Ultrasonic sensor-based water level monitoring.
- Provides relay control with upper/lower limits and visual LED level indicators.

### 🤖 Quadruped Robot (In Progress)
- Utilizes ESP32 and PCA9685 servo controller.
- Focuses on multi-servo coordination for walking mechanisms with future plans for AI-based movement control.
